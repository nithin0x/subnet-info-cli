# subnet-info

CLI tool to inspect your public/local IPs, generate CIDR variants, and print sample AWS Security Group ingress rules.

## Install locally with pipx

```bash
pipx install .
```

## Run

```bash
subnet-info
```

## Publish to PyPI

```bash
pip install build twine
python -m build
twine upload dist/*
```
